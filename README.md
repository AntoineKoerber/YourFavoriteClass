# Your Favorite Class

A professional landing page for language tutoring services. This bilingual website (English/French) showcases language learning services with a modern, responsive design.

## Features

- **Bilingual Support**: Full English and French versions of the site
- **Responsive Design**: Mobile-friendly layout using Bootstrap 5
- **Modern UI**: Clean, professional design with smooth animations
- **Service Showcase**: Highlights different lesson types (Individual, Pair, Group)
- **Pricing Information**: Clear pricing tiers with Stripe integration
- **Contact Section**: Easy-to-use contact form
- **Testimonials**: Customer reviews and success stories

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with animations
- **Bootstrap 5** - Responsive grid and components
- **Font Awesome 6** - Icon library
- **Google Fonts** - Montserrat and Roboto Slab typography

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AntoineKoerber/YourFavoriteClass.git
   ```

2. Open `index.html` in your browser to view the English version, or `index_fr.html` for French.

3. No build process required - this is a static website.

## Deployment

This is a static website that can be deployed to any web hosting service:
- GitHub Pages
- Netlify
- Vercel
- Any traditional web host

## Customization

To customize this template:
1. Replace images in `static/image/`
2. Update text content in `index.html` and `index_fr.html`
3. Modify colors and styles in `static/css/style.css`
4. Update Stripe payment links for your own account
